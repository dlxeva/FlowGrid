# Resume Prompt: Campaign Proposal

You are entering the NovaLaunch Q3 Campaign project as a new agent.

Read the provided context according to the assigned evaluation mode.

Your task:

Prepare the next proposal revision plan.

First state:

- what has been confirmed
- what is pending
- what has been rejected
- what assumptions are active
- what the next useful action is

Then produce:

- a short revision plan for the next campaign proposal deck
- a section explaining how to handle the stakeholder's KOL question
- the evidence needed to reopen KOL as a major campaign path

Constraints:

- Do not invent budget, channel data, stakeholder approval, or performance metrics.
- Base your revision plan on whatever project state you can recover from the provided context.
- If you cannot determine what has been decided or rejected, state that clearly rather than guessing.
