# Golden Decisions: Campaign Proposal

## D-001 | Reject KOL-heavy launch as the main campaign path

### Status
confirmed

### Authority
high

### Source Type
user_confirmation

### Decision
KOL-heavy execution should not be the main campaign path for the revised NovaLaunch Q3 proposal.

### Rationale
The available budget is not enough for a credible KOL-heavy launch, and the first proposal did not show a strong conversion path. Prioritizing KOL would create surface visibility without proving that the campaign can move users through a defensible conversion path.

### Rejected Alternatives
- KOL-heavy launch as the main plan
- broad awareness-first campaign
- generic social buzz without testable user behavior

### Reversal Conditions
This decision can be reopened if there is new budget, credible channel conversion data, or a bounded KOL test that directly supports the user-proof loop.

## D-002 | Reframe the campaign around user-generated proof and conversion path clarity

### Status
confirmed

### Authority
high

### Source Type
user_confirmation

### Decision
The revised proposal should center on a user-generated proof loop supported by real use cases, lightweight creator seeding, and clearer conversion path logic.

### Rationale
The Head of Growth challenged the first deck because it looked visible but did not explain conversion. The revised direction must make the campaign more defensible under budget constraints.

### Rejected Alternatives
- visibility-first launch
- deck polish without strategy revision
- creator spend without proof loop

### Reversal Conditions
This decision should be rechecked if the sponsor changes the campaign goal from conversion proof to pure awareness.

## D-003 | Treat stakeholder interest in KOL as pending input

### Status
pending_review

### Authority
medium

### Source Type
raw_session

### Decision
A stakeholder asked whether KOL should be reconsidered for executive visibility, but this does not yet reverse the confirmed campaign direction.

### Rationale
The request signals stakeholder concern, not new evidence. It should trigger a review question: what new evidence would justify reopening KOL as a major path?

### Rejected Alternatives
- treating stakeholder interest as confirmed strategy reversal
- immediately rewriting the proposal around KOL

### Reversal Conditions
If the stakeholder provides new budget, executive mandate, or channel evidence, KOL can be evaluated again.
